# Bug Bounty Scope

## In Scope
*   `api.chatflect.com` (Core API)
*   `chatflect.com` (Web Client)
*   Mobile App (Android/iOS)

## Out of Scope
*   Third-party integrations
*   Social engineering
*   DDoS attacks
*   Login bruteforce (we have rate limits)

## Rewards
| Severity | Reward Range |
| :--- | :--- |
| **Critical** | $5,000 - $10,000 |
| **High** | $1,000 - $3,000 |
| **Medium** | $100 - $500 |
| **Low** | Swag / HoF |
